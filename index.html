Find your location using
Geolocation API

Get current position, watch for changes in the devie's position and clear previous positions

With geolocation.getCurrentPosition()

You can get the device's current location.

With geolocation.watchPosition()

Registers a handler function that will be called automatically each time the position of the device changes, returning the updated location.
